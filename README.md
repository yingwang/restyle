# restyle

把一段中文改得不像机器写的。

机器写中文的毛病不在写得差，在写得太匀：每句一样长、一样满，词都挑最稳的，道理替你说尽。restyle 把这股顺劲往回收，收到汪曾祺那种写法上去，干净、平淡有味、长短句相错、落在具体的人和事上，不升华、不端着。

不是同义词替换，也不是「humanizer」。它认一个具体的声音去改：默认照作者自己的声音收，`--like` 接作家就仿那个人。

## 装

```bash
ln -s "$PWD" ~/.claude/skills/restyle
```

或者直接把这个目录拷进 `~/.claude/skills/`。

## 用

```
/restyle <贴进来的文字 | 文件路径> [--like 作家/作品] [--clean]
```

- `--like 沈从文`：仿这个作家的腔重写
- `--clean`：只给净稿，不附改动说明

具体改法和例子见 `SKILL.md`。
