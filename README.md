# restyle

把一段中文写成、或改成汪曾祺那样：平淡、干净、从容，把感情和意思都搁在具体的事物里，不抒情、不点题、不升华。句子短，可是流动，不切碎。

机器写的中文太满、太会说、什么都替你讲明白，读着顺却没味。汪曾祺正相反，什么都不点破，把滋味化在平淡里。照他的路子收一遍，那股机器气也就散了。

默认就照汪曾祺，`--like` 接别的作家就仿那个人。

## 装

```bash
ln -s "$PWD" ~/.claude/skills/restyle
```

或者把这个目录拷进 `~/.claude/skills/`。

## 用

```
/restyle <贴进来的文字 | 文件路径> [--like 作家/作品] [--clean]
```

- `--like 张爱玲`：落点换成那个作家的腔
- `--clean`：只给净稿，不附改动说明

具体路子和例子见 `SKILL.md`。
