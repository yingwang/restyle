# restyle

把一段中文改得自然、具体、舒展、流畅。不是压短，不是摘要，也不是把话切成一串短句；不要英文句式带出来的翻译腔，不要提纲腔，也不要把意思压成那种平时没人真会说的概括词。该展开就展开，写得充分一点，让句子自己往下走。

机器写的中文太会归纳，太爱替人下判断，句子圆滑，可是没有来处。restyle 要做的不是省字数，而是把空泛的判断拆回具体的人、事、动作和声音里，把该说清的地方说清，把段落写完整。长句可以用，只要主干清楚，气口顺，读下来不拧巴。汪曾祺可以借一点：平、准、从容，不端着，不摆拍；但不默认模仿他的腔。

默认保留原文自己的声音，把句子写得流畅、详细、自然。`--like` 接别的作家，才额外借那个人的高层风格特征。

## 装

```bash
ln -s "$PWD" ~/.claude/skills/restyle
```

或者把这个目录拷进 `~/.claude/skills/`。

## 用

```
/restyle <贴进来的文字 | 文件路径> [--like 作家/作品] [--clean]
```

- `--like 张爱玲`：落点换成那个作家的腔
- `--clean`：只给净稿，不附改动说明

具体路子和例子见 `SKILL.md`。
